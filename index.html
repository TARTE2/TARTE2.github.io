ModelViewer avec Three.js

ModelViewer est un petit projet interactif permettant d'importer et de visualiser des fichiers 3D au format GLB (GL Transmission Format). Utilisant la puissante bibliothèque Three.js, ce visualiseur offre une interface intuitive pour explorer des modèles 3D en temps réel. Les utilisateurs peuvent charger leurs propres fichiers GLB via une interface de sélection de fichiers, puis interagir avec le modèle en utilisant des fonctionnalités telles que le zoom, la rotation et le déplacement de la caméra.

Event Configurator avec Sortable.js

Event Configurator est un projet permettant d'éditer un fichier de configuration JSON via une interface intuitive (sujet de stage chez Mobil-inn). Utilisant la bibliothèque Sortable.js pour le drag and drop des éléments HTML, ce configurateur facilite la gestion et l'organisation dynamique des données. Bootstrap est également employé pour fournir des composants front-end modernes et réactifs.
